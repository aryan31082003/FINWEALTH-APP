# FINWEALTH-APP
Finwealth desktop web application - Manages financial information along presenting data with visual graphs

# TECH STACK
HTML - For frontend structures
CSS - for styling
Javascript - to handle client and server side logic as well as for graphical presentation
Node js - To handle backend
MongoDB - To store realtime data associated with each user
Express -  RESTFUL APIs.

# WORKFLOW EXAMPLE
-> New user sign up using his name and password
-> User logs in successfully ( with correct password and name )
-> Dashboard loads in
-> User can see total expence , total salary
-> User can add/delete individual expence / income in row
-> User can upload his photo
-> User logs out
